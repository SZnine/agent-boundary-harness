## 一、沙箱运行时（Sandbox Runtime）

### Q1.1: 沙箱隔离粒度

**问题**：被测 Agent 运行在什么隔离边界内？

**选项**：
- A. Docker 容器（进程级隔离）
- B. Python subprocess + chroot（文件系统隔离）
- C. 虚拟化（QEMU / VM）
- D. 无隔离，只记录副作用（完全依赖 fake tools）

**约束**：你的选择会影响：
- 副作用记录的可靠性
- 启动成本（v0.1 不适合重型方案）
- 是否允许真实网络请求
